I. Data Logging

i. HackerHub does log data on its users using Google Analytics. The data that is kept is not identifying and therefore can do no harm.


ii. HackerHub logs the following
- Concurrent Users
- IP Country of the user


iii. This data is not and will not be shared with any partners or affiliates.

See more on Data Logging here



II. Cryptomining

i. HackerHub uses CoinHive to mine the Monero Cryptocurrency at 10% CPU Power. This is completely optional. When using the Discord Invites the captcha also mines 256 Hashes worth of Monero. HackerHub will never mine cryptocurrencies without your consent.


ii. HackerHub does not mine cryptocurrencies if it detects the user is on a mobile device.


III. Partnership

i. The Partnership program will require users to give contact info. HackerHub does not share any of this data.


ii. A user's Partnership status can be stripped at any time with or without reason.


iii. Partners cannot share sensitive data with affiliates or other partners unless absolutely necessary.
