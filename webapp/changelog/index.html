Android Changelog

Google Play 3.1 Release Notes
- http://hacker-hub.com/ links now work ( https://hacker-hub.com / still works )
- ActionBar added
- Can now go back to the home menu with the ActionBar (instead of holding the back button)
- Can now open the current URL in another app with the ActionBar
- Can now copy the current URL to clipboard
- Also added icon overflow for smaller screens
GitHub 3.1 Release Notes
- WebView will now allow https://hacker-hub.com / and http://hacker-hub.com/ URLs
- WebView now has actionbar which reflects webpage name
- ActionBar now allows the user the go back to the previous activity with a button instead of previously holding the back button
- ActionBar now allows the user to open the current WebView URL in another app
- ActionBar now allows the user to copy the current WebView URL to clipboard
- ActionBar also has overflow icon for smaller displays
