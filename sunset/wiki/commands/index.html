2Ball

Description: Have the mystical 2Ball answer your most burning questions.
Arguments: <question>


When using the command without any arguments, 2ball, with Sunset LiteMode disabled, and without any funds in your balance, Sunset will return an error.

When using the command without any arguments, 2ball, (and with enough funds) Sunset will return a help embed.

When using the command with a question provided, 2ball <question> , the mystical 2Ball will answer your question.

If you get No as an answer you will lose $10. If you get Yes as an answer you will gain $20 ($10 in profit)


Sunset LiteMode users do not have access to Sunset Economy features and will not receive payment inquiries.

8Ball

Description: Have the mystical 8Ball answer your most burning questions.
Arguments: <question>


When using the command without any arguments,8ball, with Sunset LiteMode disabled, and without any funds in your balance Sunset will return an error.

When using the command without any arguments, 8ball, (and with enough funds) Sunset will send a help embed.

When use the command with a question, 8ball <question> , with LiteMode disabled you the mystical 8Ball will answer your question.

Depending on the 8Ball's answer you will lose $20 or gain $20 in profit.


Sunset LiteMode users do not have access to Sunset Economy features and will not receive a payment inquiry.

Announce

Description: Send announcements to other channels.
Required Permissions: MANAGE_CHANNELS KICK_MEMBERS
Arguments: <#channel> <announcement>


When using the command without any arguments, announce, you will receive an argument error.

When providing a channel, but without an announcement, announce <#channel>, Sunset will attempt to send a blank announcement to the provided channel. However, the DiscordAPI does not allow this and the announcement will not send. If a message is provided, announce <#channel> <announcement>, Sunset will send the announcement to the provided channel.


ATM

Description: Interact with the Per-Server ATM
Arguments: <withdrawal|deposit|@user> <amount>
Sections: Deposit | Withdrawal

Not LiteMode Compatible


When using the command without any arguments, atm, Sunset will show your Main Balance, your ATM Balance, and your Net Worth.

When mentioning a member on the server, atm <@user>, Sunset will show the mentioned member's Main Balance, your ATM Balance, and your Net Worth.

See Bal for more information

ATM Deposit

When attempting to deposit money without any arguments, atm deposit, you will receive an argument error.

When attempting to deposit money into your ATM Balance, atm deposit <amount>, Sunset will deposit funds from your regular Balance to your ATM Balance.


At this point the money has been deposited into your ATM Balance. You can use atm once more to check.
If you attempt to deposit more money than what you have, you will receive an error.

ATM Withdrawal

When attempting to withdraw money without any arguments, atm withdrawal, Sunset will send a help embed.

When attempting to withdraw money from your ATM Balance, atm withdrawal <amount>, your funds will be withdrawn from your ATM Balance into your regular Balance.

At this point the money has been withdrawn from your ATM Balance. You can use atm once more to check.
If you attempt to withdraw more money than what you have, you will receive an error.

Avatar

Description: View the avatar of yourself or the mentioned member.
Arguments: <@user>


When using the command without any arguments, avatar, Sunset will return a help embed.

When using the command with a mentioned member, avatar, Sunset will send you the avatar of the mentioned member in an embed.

Bal(ance)

Description: Check your or the metioned member's balance.
Not LiteMode Compatible

When using the command without any arguments, bal, Sunset will return a help embed.

When mentioning a member, bal <@user>, Sunset will send you the balance information of the mentioned member.

Ban

Description: Ban the mentioned member.
Required Permissions: BAN_MEMBERS
Arguments: <@user> <reason>

When using the command without any arguments, ban, Sunset will return a help embed.

When attempting to ban a user without reason, ban <@user>, you will be met with an error.
When attempting to ban a user with reason, ban <@user> <reason>, Sunset will ban that user and your message will be deleted.

Note: If Sunset's role is lower than the member to be banned, Sunset will return an error.
If the member is not bannable, Sunset will return an error.
If the member to banned is yourself, Sunset will return an error.
If the member to banned is Sunset, Sunset will return an error.

Base64

Description: Enocde and Decode text in Base64
Arguments: <encode|decode|recover> <text<
Sections: Encode / Decode | Recover

When using the Base64 command without any arguments, base64, Sunset will send you a help embed on the Base64 command.

Base64 Encode / Decode

For encoding or decoding a Base64 string, base64 <encode|decode>, Sunset will encode or decode your message. The result will be saved and uploaded in a text file.

To recover your last decoded or encoded string see Base64 Recover.

Base64 Recover

For recovering your last encoded or decoded text using base64 recover will upload your last encoded or decoded text file.

Your last string will be sent in a text file. This can only be done per-server.
The LiteMode version of this command sends messages without embeds (excluding file uploads)

Channel Settings

Description: Modify the metadata of the current channel.
Arguments: <option> <parameter> Sections: Name | Topic | Position | Category / Parent | Webhooks

Using the Channel Settings command without any arguments, channelsettings, will return an embed with the current channel metadata.
Note: At this time Sunset cannot edit the nsfw metadata on a channel.
Note: At this time Sunset cannot edit the type metadata on a channel.

Channel Settings Name

Using the Channel Settings command with the name argument, channelsettings name <parameter>, will change the current channel's name.
Sunset will return an error if <parameter> requirements are not met.

Channel Settings Topic

Using the Channel Settings command with the topic argument, channelettings topic <parameter>, will change the current channel's topic.
Sunset will return an error if <parameter> requirements are not met.

Position

Using the Channel Settings command with the position argument, channelsettings position <parameter>, will change the current channel's position.
Sunset will return an error if <parameter> requirements are not met.

Type

Using the Channel Settings command with the type argument, channelsettings type <parameter>, will change the current channel's type.
Sunset will return an error if <parameter> requirements are not met.

WebHooks

Using the Channel Settings command with the webhook argument (without any extra parameters), channelsettings webhook, will return a help embed on the webhook section of the Channel Settings command.

channelsettings webhook delete will delete all webhooks on the server.
Sunset will return an error if <parameter> requirements are not met.

The LiteMode version of this command will not send send message with embeds.

Console

Arguments: Arguments with this command vary per section.
Sections: Console Help | Console Sys(tem) | Console Ping | Console CMD | Cowsay Cowsay | Console Cowthink | Console Figlet


Using console without any arguments Sunset will return an error

Console Help

Using console help shows all available console commands.

Console Sys(tem)

Using console sys shows system information.

Console Ping

Arguments: <address>

Using console ping without an argument will return an error.

Using console ping with a correct address will ping the provided address.

Pinging subdirectories will not work.

Console CMD

Arguments: <command>
Requirements: <BOT_OWNER>

This command is locked to the Bot Owner, TheHacker#9367, and cannot be used by any other user.

console cmd executes commands in the Linux Terminal and returns results. It cannot be used to run CLI games or anything of that nature, only single-action commands.

Copycat

Description: Make a copycat of yourself
Arguments: <message>
Permissions: MANAGE_WEBHOOKS
Note: Due to a limit of 10 WebHooks per Guild, all copycats will self-destruct in 3 minutes. You can also use channelsettings webhooks delete to delete all webhooks on the server.

Using the copycat command with a message, copycat <message>, Sunset will create a webhook with your username and avatar.

After 3 minutes, Sunset will delete the copycat webhook to save space on webhooks.

Using this command without any arguments will send a help embed.
To delete all webhooks on the server see Channel Settings Webhooks
For Sunset LiteMode users, messages will not be sent in embeds.

DelChannel

Description: Delete the current text channel
Arguments: <reason>
Permissions: MANAGE_CHANNELS
Sections: DelChannel | MkChannel | ChannelSettings

When using the DelChannel command without arguments, delchannel, Sunset will return a help embed.

If a reason is provided for the channel deletion, delchannel <reason>, Sunset will delete the current channel (provided it has the MANAGE_CHANNELS permission.)

If Sunset does not have the MANAGE_CHANNELS permission, the channel will not be delete and Sunset will return an error.

Economy Settings

Description: Change how the Sunset economy works
Arguments: <option> <value>
Permissions: SERVER_OWNER

When using economysettings without an argument, Sunset will return Economy information for your server

To edit Economy Settings for your server use economysettings <setting>. If used successfully, Sunset will return the relevant information and give the confirmation on the setting change.
