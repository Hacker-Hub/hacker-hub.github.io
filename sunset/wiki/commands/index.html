As of Sunset v6.1 the bot has 68 commands.

2Ball

Description: Have the mystical 2Ball answer your most burning questions.
Arguments: <question>


When using the command without any arguments, 2ball, with Sunset LiteMode disabled, and without any funds in your balance you will recieve this.

When using the command without any arguments, 2ball, (and with enough funds) you will receive this.

When using the command with a question provided, 2ball <question> , you will receive this.

If you get No as an answer you will lose $10. If you get Yes as an answer you will gain $20 ($10 in profit)


Sunset LiteMode users do not have access to Sunset Economy features and will receive this.

When using the command without any arguments,8ball, with Sunset LiteMode disabled, and without any funds in your balance you will receive this.

When using the command without any arguments, 8ball, (and with enough funds) you will receive this.

Depeding on the 8Ball's answer you will lose $20 or gain $20 in profit.


Sunset LiteMode users do not have access to Sunset Economy features and will receive this.

Announce

Description: Send announcements to other channels.
Required Permissions: MANAGE_CHANNELS KICK_MEMBERS
Arguments: <#channel> <announcement>


When using the command without any arguments, announce, you will receive this.

When providing a channel, but without an announcement, announce <#channel>, Sunset will attempt to send a blank announcement to the provided channel. However, the DiscordAPI does not allow this and the announcement wont send. If a message is provided, announce <#channel> <announcement>, Sunset will send the announcement to the provided channel.


ATM

Description: Interact with the Per-Server ATM
Arguments: <withdrawal|deposit|@user> <amount>
Sections: Deposit | Withdrawal

Not LiteMode Compatible


When using the command without any arguments, atm, Sunset will show your Main Balance, your ATM Balance, and your Net Worth.

When mentioning a member on the server, atm <@user>, Sunset will show the mentioned member's Main Balance, your ATM Balance, and your Net Worth.

ATM Deposit

When attempting to deposit money without any arguments, atm deposit, you will receive this.

When attempting to deposit money into your ATM Balance, atm deposit <amount>, you will receive this.


At this point the money has been deposited into your ATM Balance. You can use atm once more to check.
If you attempt to deposit more money than what you have, you will receive an error.

ATM Withdrawal

When attempting to withdraw miney without any arguments, atm withdrawal, you will receive this.

When attempting to withdraw money from your ATM Balance, atm withdrawal <amount>, you will receive this.

At this point the money has been withdrawn from your ATM Balance. You can use atm once more to check.
If you attempt to withdraw more money than what you have, you will receive an error.

Avatar

Description: View the avatar of yourself or the mentioned member.
Arguments: <@user>


When using the command without any arguments, avatar, you will receive this.

When using the command with a mentioned member, avatar, Sunset will send you the avatar of the mentioned member in an embed.

Bal(ance)

Description: Check your balance or the balance of the mentioned member.
Not LiteMode Compatible

When using the command without any arguments, bal, you will receive this.

When mentioning a member, bal <@user>, Sunset will send you the balance information of the mentioned member.

Ban

Description: Ban the mentioned member.
Required Permissions: BAN_MEMBERS
Arguments: <@user> <reason>

When using the command without any arguments, ban, you will receive this.

When attempting to ban a user without reason, ban <@user>, you will be met with an error.
When attempting to ban a user with reason, ban <@user> <reason>, Sunset will ban that user and your message will be deleted.

Note: If Sunset's role is lower than the member to be banned, Sunset will return an error.
If the member is not bannable, Sunset will return an error.
If the member to banned is yourself, Sunset will return an error.
If the member to banend is Sunset, Sunset will return an erorr.
