ABOUT SUNSET

Sunset is the Swiss Army Knife of Discord Bots

Sunset has a ton of features ranging warn ban kick and much more. Sunset is also being hosted 24/7 with very little downtime for updates.

Sunset

Sunset is programmed using NodeJS with the powerful DiscordJS library. Sunset's source code is also available on GitHub for anyone to view and contribute code.

Commands

Below are all of the commands Sunset has to offer.


Commands documented here are for the latest version of Sunset not the latest source code.

Information

help | Get documentation on all available commands for SunsetLiteMode Compatible
litemode | Activate Sunset LiteModeLiteMode Compatible
ping | Ping the Discord API and Sunset LiteMode Compatible
info | Get info on Sunset LiteMode Compatible

Server Info

inspect | Inspect many types of URLs LiteMode Compatible
serversettings | Change how Sunset functions for your server LiteMode Compatible
invite | Get an invite to the server for the provided channel LiteMode Compatible
serverinfo | View the current server/guild's info LiteMode Compatible
avatar | View the avatar of yourself or of the mentioned member LiteMode Compatible
profile | View the profile of yourself or of the mentioned member LiteMode Compatible

Entertainment

translate | Translate the provided messages from English to the provided Language LiteMode Compatible
cleverbot | Chat with CleverBot through the cleverbot.io library LiteMode Compatible
emojify | Emojify the provided message if possible LiteMode Compatible
cowthink | Returns the provided message in a cowthink code block LiteMode Compatible
cowsay | Returns the provided message in a cowsay code block LiteMode Compatible
fliptext | Flips the provided message LiteMode Compatible
figlet | Figletizes the provided message LiteMode Compatible
say | Returns the message provided in an embed LiteMode Compatible
2ball | Have your questions answered by the mystical 2ball LiteMode Compatible
8ball | Have your questions answered by the mystical 8ball LiteMode Compatible
copycat | Make a copycat of yourself
timer | Set a timer for a maximum of 12 hours LiteMode Compatible
urban | Search the Urban Dictionary for the provided word LiteMode Compatible
dictionary | Search the OxFord Dictionary for the provided word LiteMode Compatible
google | Search Google with the provided search term Note: some search terms may only work in a channel marked as NSFW LiteMode Compatible
coinflip | Flip a coin LiteMode Compatible
roll | Roll a die LiteMode Compatible

Economy

atm | Interact with the Per-Server ATM
bal | Check the balance of yourself and other members
pay | Pay the provided member; You will need to have sufficient funds to complete the payment
freemoney | Earn freemoney, the payout is randomized between $1 and $50; this can be changed with the serversettings command
lottery | Buy a lottery ticket Ticket Prices and Payouts are randomized. This can be changed with the serversettings command
rob | Rob the mentioned user; Chances of successfully robbery are randomized and can be changed with the serversettings command
clearmoney | Go off the grid and clear your entire balance and ATM balance

Content Generation

binary | Encode and Decode the provided message in binary LiteMode Compatible
url | Shorten or Expand the provided URL using the tinyurl URL Shortener LiteMode Compatible
base64 | Encode and Decode the provided message in Base64 LiteMode Compatible
morsecode | Encode and Decode the provided message in Morse Code LiteMode Compatible
qrcode | Encode the provided message in a QR Code LiteMode Compatible
lastqr | Recover the most recent QR Code generated if there is one LiteMode Compatible
createtxt | Create a text file with the provided message LiteMode Compatible
lasttxt | Recover the most recent Text File generated if there is one LiteMode Compatible

Moderation

warn | Warn the mentioned member
warnings | Check the warnings of the mentioned user if there are any
rules | View the rules of the server; These rules can be set using rules set LiteMode Compatible
channelsettings | Modify the metadata of the current channel LiteMode Compatible
mute | Mute the mentioned member
unmute | Unmute the mentioned member if they are muted
softban | SoftBan the mentioned member Just bans then unbans the mentioned member LiteMode Compatible
ban | Ban the mentioned member LiteMode Compatible
unban | Unban the mentioned member LiteMode Compatible
kick | Kick the mentioned member LiteMode Compatible
purge | Bulk Delete up to 200 messages in the current channel LiteMode Compatible
mkchannel | Make a text, voice, or category channel LiteMode Compatible
delchannel | Delete the current text channel LiteMode Compatible
setnick | Set the nickname of the mentioned member LiteMode Compatible

Other

randomcolor | Get a random color in Hex, RGB, LAB, and CMYK LiteMode Compatible

Owner-Only Commands

jsexec | Execute NodeJS through eval Locked to Owner for obvious reasons
flush | Flush parts of Sunset LiteMode Compatible
settings | Change the settings of Sunset LiteMode Compatible
setmoney | Set the balance of the mentioned member
