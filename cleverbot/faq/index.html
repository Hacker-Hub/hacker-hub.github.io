  • CleverBot doesn't respond to my messages, how can I fix this?

    If CleverBot is not responding to your messages, please complete the actions below;
    -Rebind CleverBot to the current channel (The new database implemented in v1.4 may not play nicely with older files)
    -Make sure CleverBot has permissions to send messages
    -If all else fails kick and reinvite CleverBot.

    If none of these options work please contact me at skylarmccauley864@gmail.com

  • How do I upvote CleverBot?

    Upvoting CleverBot is as easy as going to the Discord Bot List page. You can also use @CleverBot vote to get a link (and soon upvote).

  • Why is CleverBot so slow?

    Due to the recent database migration, CleverBot has to check whether your server is using the old DB files or the new DB files. This process can be sped up by making sure you have rebinded your channels.

  • Why does CleverBot keep telling me to rebind my channel?

    In the v1.4 update, CleverBot has started writing new Database files that will be more utilized in the future making the old ones obsolete. This migration also makes it so that users have to rebind their channels in order to keep using the bot without interuptions. In v1.5, users who have still neglected to rebind their channels will no longer be able to use CleverBot until they rebind.

  • Leveling System

  • How do I level up?

    To gain eXPerience and level up, you just need to interact with CleverBot. The more you interact the higher your XP will be. Once you get 20,000 XP, you'll gain one level and be booted back to 0 XP.

  • How do I get Stars?

    Once you've reached 20,000 XP and Level 500, you can use the prestige command to gain one Star. After this, you'll be booted back to 0 XP and Level 0. The maximum amound of Stars you can get is 999, so keep on grinding!

  • Data Logging

  • How much data does CleverBot log?

    CleverBot logs the following data for quality assurance (such as fishing out issues with the bot);
    -User IDs and DiscordTags#0000
    -Messages and Message IDs
    -Channel Names and Channel IDs
    -Guild Names and Guild IDs

    This is also viewable in the CleverBot disclaimer accessible by using @CleverBot disclaimer

  • Why does CleverBot log data?

    CleverBot logs user data, so it can be made better in the future. It is also logging user data for research purposes.

  • How long does CleverBot keep logged data?

    CleverBot is not set to delete any logged data, however in the future some logs may be delete to conserve space.
