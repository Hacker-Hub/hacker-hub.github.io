What's New!

CleverBot v1.4

New Features

-Added the ability for CleverBot to ignore messages that start with <ignore>
-Added an option to send support to the developer (intended use for complaints, suggestions, or just a pat on the back). User & Guild IDs are logged to monitor spam because you're all a bunch of skids.
-CleverBot now sends a confirmation message in the newly bound channel (just in-case you were confused)
-New database system! Existing users will need to rebind channels before September 22nd
-Now runs on DiscordJS v11.4

Removed Features

-Removed Interaction Count due to floating points getting too big and resetting. Might make a return in the future.

Bug Fixes

-Fixed a bug that caused CleverBot to not respond to mentions when giving a nickname
-Fixed an error in the disclaimer command where the User ID logging string would be shown twice.
