Getting Started

Getting Started

Warning! There is an issue with CleverBot that has been overlooked. Do not give CleverBot a nickname, as it will not respond to mentions. This will be addressed in a future update.

There are some settings you'll need to configure before you start chatting with CleverBot.
Permissions: MANAGE_CHANNELS This permission is only required to bind CleverBot to a channel.
Prefix: A mention


Binding CleverBot to a channel is as simple as @CleverBot bind <#channel>. If the channel provided is a: 17 or 18 digit Snowflake or a #channelmention and a Channel that exists, CleverBot will bind itself to that channel.
To chat with CleverBot from here on out just send a regular message without the prefix. If you would like to change the bounded channel later on, just use @CleverBot bind <#channel> once more.
