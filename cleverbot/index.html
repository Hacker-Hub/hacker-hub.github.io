Getting Started

Getting Started

There are some settings you'll need to configure before you start chatting with CleverBot.
Permissions: MANAGE_CHANNELS This permission is only required to bind CleverBot to a channel.
Prefix: A mention


Binding CleverBot to a channel is as simple as @CleverBot bind <channelID>. If the Channel ID provided is a: 17 or 18 digit Snowflake and a Channel that exists, CleverBot will now bind itself to that channel.
To chat with CleverBot from here on out just send a regular message without the prefix. If you would like to change the bounded channel later on, just use @CleverBot bind <channelID> once more.
