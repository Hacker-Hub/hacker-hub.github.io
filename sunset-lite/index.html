SUNSET LITE

ABOUT SUNSET LITE

Sunset Lite is the Discord Bot made just for fun.

Sunset Lite is a simpler version of Sunset. Sunset Lite doesn't rely on embeds to get show your information. It also doesn't suffer from as much latency as Sunset. If you like having a bot that's fun to keep around, Sunset Lite is perfect for you.

Sunset


Invite Sunset Lite

Sunset is programmed using NodeJS with the powerful DiscordJS library. Sunset's source code is also available on GitHub for anyone to view and contribute code.

Commands and Features

Commands

Below are all of the commands Sunset Lite has to offer.


Information

help | Get documentation on all available commands for Sunset
ping | Ping the Discord API and Sunset
info | Get info on SunsetUPDATED
usage | Get the CPU and RAM usage of Sunset's process
uptime | Get the uptime for Sunset
invite | Get an invite to the server for the provided channel
serverinfo | View the current server/guild's info
avatar | View the avatar of yourself or of the mentioned member
profile | View the profile of yourself or of the mentioned member

Entertainment

translate | Translate the provided messages from English to the provided Language
cleverbot | Chat with CleverBot through the cleverbot.io library
emojify | Emojify the provided message if possible
cowthink | Returns the provided message in a cowthink code block
cowsay | Returns the provided message in a cowsay code block
fliptext | Flips the provided message
figlet | Figletizes the provided message
say | Returns the message provided in an embed
2ball | Have your questions answered by the mystical 2ball
8ball | Have your questions answered by the mystical 8ball
timer | Set a timer for a maximum of 12 hoursUPDATED
urbandictionary | Search the Urban Dictionary for the provided wordREMOVED FROM 3.2 USE URBAN
urbandictionary | Search the Urban Dictionary for the provided wordUPDATED 3.2
dictionary | Search the OxFord Dictionary for the provided word
google | Search Google with the provided search term Note: some search terms may only work in a channel marked as NSFW
coinflip | Flip a coin
roll | Roll a die

File Generation

binary | Encode and Decode the provided message in binary
url | Shorten or Expand the provided URL using the goo.gl URL Shortener
base64 | Encode and Decode the provided message in binary
qrcode | Encode the provided message in a QR Code
lastqr | Recover the most recent QR Code generated if there is one
createtxt | Create a text file with the provided message
lasttxt | Recover the most recent Text File generated if there is one

Moderation

warn | Warn the mentioned member
warnings | Check the warnings of the mentioned user if there are any
rules | View the rules of the server; These rules can be set using rules set
channelsettings | Modify the metadata of the current channel
mute | Mute the mentioned member
unmute | Unmute the mentioned member if they are muted
ban | Ban the mentioned member
unban | Unban the mentioned member
kick | Kick the mentioned member
purge | Bulk Delete up to 200 messages in the current channel
mkchannel | Make a text voice or category channel
delchannel | Delete the current text channel
setnick | Set the nickname of the mentioned member

Other

randomcolor | Get a random color in Hex RGB LAB and CMYK
announcement | Check the current announcement

Owner-Only Commands

jsexec | Execute NodeJS through eval Locked to Owner for obvious reasons
flush | Flush parts of Sunset
settings | Change the settings of Sunset
