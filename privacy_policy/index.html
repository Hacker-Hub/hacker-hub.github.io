I. Data Logging

i. This website does log data on its users using Google Analytics. The data is not identifying or incriminating as it is anonymous.


ii. This website logs the following
- Concurrent Users
- IP Country of the user (This data is never shared or transmitted)
- Device of the user
- Session time
- Time of day page was accessed by the user
- What pages were accessed by the user
- User Retention
- Operating System
- Browser
- ISP


iii. This data is not and will not be shared with any partners or affiliates. We do not wipe any logs from our servers or Google Analytics profiles, however, we will delete data that we no longer have any user for. Data is kept on the HackerHub Google Analytics profile and also is occasionally downloaded for quality assurance and quarterly reports. Any data processor that is affiliated with us will be forced to sign a Non-Disclosure Aggreement (NDA) before being able to process our logs.


iv. User data is logged for viewing what users like best and how we can make HackerHub better. In no way, shape, or form will we change that.

If any user would like to request access to personal information that we have kept you can visit the HackerHub Data Request page for more information.

v. Indentifying Discord User Data is logged through HackerHub bots such as CleverBot and Sunset.
Logged Data includes; - User IDs
- Channel IDs
- Server/Guild IDs
- Message IDs
- Whether the message was ignored or not (CleverBot)

CleverBot also logs your conversations.
